fieldMult :: Word8 -> Word8 -> Word8 Source #

We only need to implement multiplication by 2 and 3 in GF(2^8) for encryption, 9, 11, 13, and 14 for decryption. Multiplication by 2 is equivalent to bit shifting by one and adding (xor) 0x1b (27) if the high bit was set. 3 x b = (2 xor 1) b = (2 x b) xor b
